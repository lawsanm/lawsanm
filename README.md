### 💎 LauxUp — The Modern Moodle Experience
> *A Chrome extension that transforms the Moodle LMS into a premium, distraction-free workspace*

| Feature | Details |
|---------|---------|
| 🌙 Deep Dark Mode | Pre-injected at `document_start` for zero white flash |
| 📌 Notion-style Sidebar | Persistent, collapsible navigation with courses & assignments |
| 📊 Intelligent Dashboard | Redesigned greeting, progress rings, and beautiful course cards |
| ⌨️ Power Shortcuts | `D` Dashboard · `C` Courses · `A` Assignments · `[` Sidebar |
| 🏗️ Architecture | Manifest V3 · Modular CSS Design System · DOM-first approach |

**Tech:** `JavaScript` `CSS Design Tokens` `Chrome Extension API (MV3)` `DOM Manipulation`

> *Plan smarter trips worldwide with cost intelligence, itinerary generation, and fair budget splitting*

| Feature | Details |
|---------|---------|
| 🎬 Cinematic UI | Glassmorphism "water glass" design with dynamic background slideshows |
| 💰 Budget Engine | Intelligent cost estimation across Budget / Mid / Luxury tiers |
| 🗺️ Itinerary Generator | Automated daily plans with integrated destination data |
| 🔒 Production Security | JWT auth, bcrypt, rate limiting, CORS, security headers |
| 🐳 Docker-Ready | Full-stack orchestration with `docker-compose` |

**Tech:** `React` `Vite` `Tailwind CSS` `FastAPI` `SQLAlchemy` `Docker` `Google Maps API`
